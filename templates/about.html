About the Reinforcement Learning Simulator

This simulator was created by Michael Roberts as a personal project while taking a graduate level Reinforcement Learning class. The main goal was to implement new things learned in a deployed web environment. This simulator is for educational purposes for users see how different policies learned using different parameters change the behavior of the agent during the training process. There are tons of different environments that can be added or used for Reinforcement Learning but the goal was to create something simple. The environments and coding documentation referenced in this project came from the python package gymnasium. Feel free to reach out to me on LinkedIn!
